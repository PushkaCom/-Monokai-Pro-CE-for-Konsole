# Monokai Pro (Community Edition) for [Notepad++](https://notepad-plus-plus.org/)

## Installation instructions

Clone from Github:

```zsh
git clone https://github.com/PushkaCom/Monokai-Pro-CE-for-Konsole/
```

### Activating theme - KDE

1. Search for "Monokai Pro CE" in: settings -> configure konsole -> profiles -> edit -> appearance -> add new
2. or install from https://store.kde.org/p/2355894/

### Activating theme

1. Download the file Monokai-Pro-CE.colorscheme
2. move to: ~/.local/share/konsole/ - where ~ is your home folder /home/username/
3. Activate theme in settings -> configure konsole -> profiles -> edit -> appearance -> select from the list
4. Enjoy!
